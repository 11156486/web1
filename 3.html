태그의 종류

- br : 줄바꿈을 하고싶을 때 사용, br태그는 여는 태그만 사용한다.
- p : 문단을 구분하고 싶을 때 사용
- h1~6 : 제목을 구분하고 싶을 때 사용하며, 중요도가 높은 순서대로 오름차순으로 쓰인다.

태그의 적절한 사용

웹 페이지 상으로 보기에 별 차이가 없어 보이더라도 태그를 그 의미에 맞게 사용하는 것은 대단히 중요하다. 그중 특별한 예를 하나 들자면, 웹브라우저는 수많은 웹페이지 중에서 어떤 키워드를 제목으로 썼는지, 본문 내용으로 썼는지를 비교하여 제목으로 사용한 경우를 더 중요한 정보로 판단하여 검색하여 나온 웹페이지 목록 중 상위에 두게끔 설계되어 있다. 이같은 작은 부분이 결과적으로 얼마나 큰 영향을 줄지는 누구나 알 것이라 생각한다.
